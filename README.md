# Meli Front

## Description
Is an application for searching items to buy which has three principal views: search, results list and details items,
It is also conected to a backend application "MELI-SERVER"

## Tecnologies

- React js
- Styled components
- Axios
- vite
- React router dom
- React spinners

## Installation Guide
### Prerequisites
* Node version: "16.14.0"
* Setup .env file

### Install
* npm install

### Run app
* npm run dev
